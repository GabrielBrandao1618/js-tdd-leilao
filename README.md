# TDD js example - leilão

Exemplo de TDD criado em Javascript utilizando leilão como exemplo

## Instruções

### 1. Instalar as dependências

yarn

```bash
yarn install
```

npm

```bash
npm install
```

pnpm

```bash
pnpm install
```

### 2. Iniciar os testes 

yarn
```bash
yarn test
```
npm
```bash
npm run test
```
pnpm
```bash
pnpm run test
```
